# Private Browser

A simple web-based private browser that routes traffic through a proxy to hide your IP address and prevent tracking by external IPs. Built with Node.js, Express, and a basic HTML/CSS/JavaScript frontend.

## Features
- **Proxy Support**: Route web requests through a proxy to mask your IP.
- **No Cookies**: Disables cookies to prevent tracking.
- **Session Clearing**: Option to clear session data (extendable for future use).
- **Simple Interface**: Lightweight UI for entering URLs and managing proxy settings.

## Prerequisites
- Node.js (v14 or higher)
- A proxy server (free or paid, e.g., from `free-proxy-list.net` or services like NordVPN, ProxyMesh)
